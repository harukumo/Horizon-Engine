## Unit System

Mass unit: Kilogram (kg)

Time unit: Second

Distance unit: Meter

Sound unit: Decibel (dB)

## Coordinate System
Horizon Engine uses a right-handed coordinate system.


                +z
                |
                |
                |
                |
                |
                ——————————————> +y
               /
              /
             /
            /
          +x

If the camera does not have any rotation, it will look in the positive direction of the Y axis by default

        +z
        |    +y
        |   /
        |  /
        | /
        |/
        ——————————————> +x
